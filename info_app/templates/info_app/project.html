Project Overview

This group project is designed to help you apply the concepts learned throughout the course to a real-world scenario. You'll work in teams to design, develop, and present a complete software solution.

Project Brief

Your team will develop a web-based application for managing course resources and student submissions. The system should include:

  • User authentication and role-based access control
  • Course material organization and distribution
  • Assignment submission and tracking
  • Communication features (announcements, messaging)
  • Progress tracking and reporting

Technology Stack

You are free to choose your technology stack, but we recommend:

  • Frontend: HTML5, CSS3, JavaScript (React/Vue.js recommended)
  • Backend: Node.js, Python (Django/Flask), or Java (Spring)
  • Database: MySQL, PostgreSQL, or MongoDB
  • Version Control: Git with GitHub/GitLab

Submission Guidelines

What to Submit

  • Complete source code with proper documentation
  • Database schema and sample data
  • User manual and technical documentation
  • Presentation slides and demonstration video
  • Team contribution report

Submission Format

All submissions should be made through the LMS. Please follow these guidelines:

  • Code should be submitted as a GitHub/GitLab repository link
  • Documents should be in PDF format
  • Videos should be uploaded to a cloud service (Google Drive, OneDrive) with appropriate sharing permissions
  • Name your files using the format: TeamName_MilestoneX_Description.ext

Grading Criteria

  • Functionality and completeness (40%)
  • Code quality and documentation (25%)
  • User interface and experience (15%)
  • Presentation and documentation (20%)
